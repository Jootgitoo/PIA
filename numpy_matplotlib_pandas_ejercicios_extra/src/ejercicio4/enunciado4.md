## Ejercicio 4: Análisis de alturas

1. Genera con **NumPy** 50 alturas aleatorias (en cm) entre 150 y 200.
2. Crea un DataFrame con una columna `altura`.
3. Calcula:
    - Media
    - Mediana
    - Desviación estándar
4. Añade una columna `grupo`:
    - `"baja"` si < media
    - `"alta"` si ≥ media
5. Dibuja:
    - Histograma de alturas
    - Línea vertical indicando la media
