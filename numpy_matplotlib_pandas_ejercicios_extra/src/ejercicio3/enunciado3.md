## Ejercicio 3: Ventas de una tienda

1. Genera con **NumPy**:
    - Ventas diarias aleatorias durante 15 días
    - Valores entre 100 y 1000 €
2. Crea un DataFrame con:
    - `dia`
    - `ventas`
3. Añade:
    - Una columna `ventas_acumuladas`
    - Una columna `objetivo` (True si ventas ≥ 500)
4. Calcula:
    - Total de ventas
    - Porcentaje de días con objetivo cumplido
5. Representa:
    - Gráfico de barras de ventas diarias
    - Gráfico de líneas de ventas acumuladas
