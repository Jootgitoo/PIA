## Ejercicio 5: Comparación de dos grupos

1. Genera con **NumPy** dos arrays de 30 valores cada uno (grupo A y B).
2. Crea un DataFrame con:
    - `valor`
    - `grupo`
3. Calcula la media de cada grupo usando Pandas.
4. Representa con Matplotlib:
    - Un gráfico de barras con la media por grupo
    - Un boxplot para comparar distribuciones

5. 
    - Guarda uno de los DataFrames en un archivo CSV.
    - Vuelve a cargarlo y comprueba que los datos son correctos.
    - Repite alguna gráfica usando los datos cargados.
