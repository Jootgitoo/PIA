## Ejercicio 1: Análisis básico de notas de una clase

1. Usa **NumPy** para generar un array de 20 notas aleatorias entre 0 y 10.
2. Crea con **Pandas** un DataFrame con:
    - `alumno` → Alumno 1, Alumno 2, …
    - `nota`
3. Añade una columna `aprobado` (`True` si nota ≥ 5).
4. Calcula:
    - Nota media
    - Nota máxima y mínima
5. Representa con **Matplotlib**:
    - Un histograma de las notas
    - Una línea horizontal indicando la nota media
